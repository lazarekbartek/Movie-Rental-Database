# Movie Rental Database

## Overview

This project presents the design and implementation of a relational database for a movie rental system using Microsoft SQL Server. The database was designed to model the core operations of a rental business and support efficient data storage and analysis.

## Technologies

- Microsoft SQL Server
- SQL

## Features

- Relational database design
- Primary and foreign key implementation
- Database normalization
- SQL queries for data retrieval and analysis

## Database Structure

The database includes tables for:

- Customers
- Movies
- Categories
- Actors
- Rentals
- Inventory
- Payments
- Staff

## Learning Outcomes

This project allowed me to develop practical skills in:

- Relational database design
- SQL programming
- Data modeling
- Database normalization
- Query optimization
